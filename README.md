# Sac AI Hub

Community resource hub for the Sacramento AI Collective — members, events, tools, and links, all maintained by the community.

**Live site:** https://zenpierce27.github.io/sac-ai-hub/

---

## What's here

| Section | Data file |
|---|---|
| Members | `src/data/members.json` |
| Events | `src/data/events.json` |
| Resources | `src/data/resources.json` |

Want to add yourself, an event, or a resource? See [CONTRIBUTING.md](./CONTRIBUTING.md).

---

## Run locally

```bash
git clone https://github.com/zenpierce27/sac-ai-hub.git
cd sac-ai-hub
npm install
npm run dev
```

Site runs at `http://localhost:4321`

---

## Stack

- [Astro](https://astro.build) — static site framework
- Plain JSON for data
- GitHub Actions for CI/CD
- GitHub Pages for hosting

No database. No backend. No cost.
